# Excel Automation with Python

This project automates Excel report processing using Python.

## What does this project do?
- Reads multiple Excel files from a folder
- Merges them into a single dataset
- Creates a summary report (total, average)
- Exports the final report to a new Excel file

## Who is this for?
- Small businesses
- Accountants
- Schools
- Anyone who works with repetitive Excel tasks

## Technologies Used
- Python
- pandas
- openpyxl

## How to use
1. Put your Excel files into the input folder
2. Run the Python script
3. Get an automatic summary Excel report
